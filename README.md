# 🌱 EcoLearn

EcoLearn is an interactive learning platform designed to educate users about **environmental conservation, sustainability, and eco-friendly practices**. Built with **React** for the frontend and **Node.js/Express** for the backend.  

---

## 🚀 Features

- 🌍 Learn about environmental topics through interactive lessons  
- 📊 Track your learning progress  
- 🎯 Quizzes to test your knowledge  
- 📝 Notes and resources for eco-friendly practices  
- 🔒 Secure authentication and personalized profiles  

---

## 💻 Tech Stack

**Frontend:**  
- React ⚛️  
- TailwindCSS 🎨  
- React Router 📌  
- Framer Motion ✨  

**Backend:**  
- Node.js & Express ⚡  
- MongoDB 🗄️ (via Mongoose)  
- JWT Authentication 🔐  
- Node-Cron for scheduled tasks ⏱️  
- Axios for API requests 📡  

---

## 🛠️ Installation

1. Clone the repo:  
```bash
git clone https://github.com/<your-username>/EcoLearn.git
cd EcoLearn

Install dependencies:

# Frontend
cd frontend
npm install

# Backend
cd ../backend
npm install

Create .env file in backend/ with:
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000

Run the project:

# Backend
cd backend
npm run dev

# Frontend
cd ../frontend
npm run dev
Open your browser at http://localhost:5173
